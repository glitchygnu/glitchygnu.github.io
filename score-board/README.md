# 🏐 Ultimate Score Board - Volleyball Score Keeper

A fully-featured, responsive, and visually stunning scoreboard website perfect for volleyball, badminton, tennis, or any sport that requires score tracking with serve rotation.

## ✨ Features

### Core Functionality
- **Two Large Score Fields** - Tap anywhere on left/right field to increment score
- **Serve Tracking System** - Automatically tracks serve changes every 2 total points
- **Undo Functionality** - Each field has an undo button to revert accidental points
- **24 Point Max** - Game stops when a player reaches 24 points
- **Reset Button** - Start a new set at any time

### Visual Features
- **4 Unique Visual Modes**:
  - ✨ **Normal** - Clean and professional design
  - 🌈 **Vibrant** - Bright, energetic colors with animations
  - 💻 **Hacker** - Matrix-style green on black theme
  - 💎 **Liquid Glass** - Frosted glass effect with depth

- **Customizable Colors** - Choose from predefined colors for each field
- **Rich Animations** - Point explosions, serve transitions, win celebrations
- **Responsive Design** - Works perfectly on desktop, tablet, and mobile

### User Experience
- **Touch & Mouse Support** - Works on all devices
- **Haptic Feedback** - Vibrates on mobile devices when scoring
- **Toast Notifications** - Visual feedback for all actions
- **Keyboard Shortcuts** - Power users can use keyboard controls
- **Persistent Settings** - Saves preferences in localStorage

## 🎮 How to Use

### Basic Controls
- **Tap/Click on Left/Right Field** - Add 1 point to that player
- **Undo Button** - Remove the last point from that field
- **Reset Button** - Reset the entire set
- **Settings Gear** - Open customization menu

### Keyboard Shortcuts
- `←` (Left Arrow) - Add point to left player
- `→` (Right Arrow) - Add point to right player
- `Ctrl/Cmd + Z` - Undo last point
- `Ctrl/Cmd + R` - Reset game
- `Ctrl/Cmd + Shift + S` - Open settings
- `Ctrl/Cmd + 1-4` - Switch visual modes
