# Score Board - Volleyball Score Keeper

A responsive, touch-friendly scoreboard website for volleyball/sports scoring with serve tracking.

## Features

- **Two large score fields** - Tap anywhere on left/right field to increment score
- **Serve tracking** - Serve changes every 2 total points, select who starts
- **Undo functionality** - Each field has an undo button to revert accidental points
- **Max 24 points** - Game stops when a player reaches 24
- **4 Visual Modes** - Normal, Vibrant, Hacker, Liquid Glass
- **Customizable colors** - Choose from predefined colors for each field
- **Responsive design** - Works on desktop, tablet, and mobile
- **Persistent settings** - Saves preferences in localStorage
